# Конвертер списков абитуриентов УлГТУ

Консольное приложение преобразует PDF-файл «Список деканам» в удобные Excel-файлы.
Для каждого выбранного факультета создается отдельный список абитуриентов.

Программа умеет:

- объединять повторные заявления одного абитуриента по Email;
- показывать бюджетные приоритеты по направлениям;
- собирать дополнительные квоты в одном столбце;
- подсвечивать приоритеты 1-3 цветом;
- формировать лист со статистикой;
- обновлять ранее созданный Excel-файл;
- сохранять ручные записи из столбца `Примечание` при обновлении.

## Быстрый запуск в Windows

Этот способ подходит обычному пользователю. Устанавливать Python не нужно.

### 1. Скачайте программу

1. Откройте страницу [последнего релиза](https://github.com/antonskalkin73/ulstu-reception-convertor/releases/latest).
2. В разделе **Assets** скачайте файл:

   `ulstu-reception-convertor.exe`

Это готовая программа. Распаковывать файл и устанавливать Python не нужно.

### 2. Подготовьте отдельную папку

Создайте удобную папку внутри папки своего пользователя, например в `Документах`. Для работы в такой папке обычно не требуются права администратора.

Пример для пользователя `Иван`:

```text
C:\Users\Иван\Documents\Приемная_кампания
```

Переместите скачанный EXE в эту папку:

```text
C:\Users\Иван\Documents\Приемная_кампания\
└── ulstu-reception-convertor.exe
```

При первом запуске программа сама создаст рядом файл настроек `direction_short_names.ini`.

### 3. Запустите программу

Дважды щелкните по файлу:

```text
ulstu-reception-convertor.exe
```

Если Windows показывает предупреждение о неизвестном издателе, проверьте, что файл скачан из Releases этого репозитория. Затем выберите **Подробнее** и **Выполнить в любом случае**.

### 4. Укажите факультет

Первый вопрос программы:

```text
Факультет:
```

Введите короткий код факультета, например:

```text
ФИСТ
```

В этом случае будет создан файл только для ФИСТ.

Чтобы создать отдельные файлы для всех факультетов из PDF, оставьте строку пустой и нажмите Enter или введите:

```text
0
```

Если вы обновляете существующий Excel-файл, можно оставить факультет пустым. Программа попробует определить его по листу `Статистика`.

### 5. Укажите исходный PDF

Следующий вопрос:

```text
Введите путь к PDF-файлу:
```

Пример:

```text
C:\Users\Иван\Documents\Приемная_кампания\СПИСОК_ДЕКАНАМ_09_07_2026.pdf
```

Путь можно вставить с кавычками:

```text
"C:\Users\Иван\Documents\Приемная_кампания\СПИСОК_ДЕКАНАМ_09_07_2026.pdf"
```

Получить полный путь к файлу можно через Проводник:

1. Найдите PDF-файл.
2. Зажмите `Shift` и щелкните по файлу правой кнопкой мыши.
3. Выберите **Копировать как путь**.
4. Вставьте путь в окно программы сочетанием `Ctrl+V` и нажмите Enter.

Скопированный путь может содержать кавычки по краям — программа удалит их автоматически. Также можно перетащить PDF мышкой в окно консоли, чтобы подставить его полный путь.

### 6. Создайте новый файл или обновите старый

Последний вопрос:

```text
Путь к существующему XLSX (Enter - создать новый):
```

#### Создание нового списка

Оставьте строку пустой и нажмите Enter.

Новые файлы появятся в папке `Результаты` рядом с программой. Имя файла имеет вид:

```text
Абитуриенты_<Факультет>_<Дата>.xlsx
```

Пример:

```text
Абитуриенты_ФИСТ_2026-07-09.xlsx
```

Если файл с таким именем уже существует, программа добавит к новому имени время и не перезапишет предыдущий результат.

#### Обновление существующего списка

Введите полный путь к ранее созданному XLSX:

```text
C:\Users\Иван\Documents\Приемная_кампания\Результаты\Абитуриенты_ФИСТ_2026-07-09.xlsx
```

Перед обновлением закройте этот файл в Excel. Открытый файл Windows может запретить программе заменить.

Во время обновления:

- список абитуриентов и приоритеты заново читаются из нового PDF;
- статистика пересчитывается;
- значения столбца `Примечание` переносятся по Email;
- существующий XLSX обновляется на месте;
- в консоли выводится количество измененных приоритетов, новых абитуриентов и абитуриентов, которых больше нет в новом PDF.

Один существующий XLSX относится к одному факультету. Для обновления файлов нескольких факультетов запускайте программу отдельно для каждого файла.

### 7. Дождитесь завершения

Обработка большого PDF может занять некоторое время. После завершения программа покажет:

- количество найденных факультетов;
- количество конкурсных групп и заявлений;
- количество уникальных абитуриентов;
- полный путь к созданному или обновленному XLSX;
- результат сравнения приоритетов при обновлении.

Консоль не закроется автоматически. Прочитайте результат и нажмите любую клавишу для выхода.

## Настройка сокращений направлений

Файл `direction_short_names.ini` должен находиться рядом с EXE. Его можно открыть обычным Блокнотом.

Пример:

```ini
[ФИСТ]
Информатика и вычислительная техника = ИВТ
Программная инженерия = ПИ
```

Правила:

1. В квадратных скобках указывается код факультета.
2. Слева от знака `=` указывается полное название направления из PDF.
3. Справа указывается желаемое короткое название столбца.
4. Для другого факультета добавьте новую секцию, например `[ЭФ]`.
5. Если сокращение не задано, программа сформирует его автоматически.

Если файла `direction_short_names.ini` нет, приложение создаст пример при первом запуске.

## Содержимое Excel-файла

В книге создаются три листа.

### `ReadMe`

Краткая памятка по работе с конкретной выгрузкой.

### `Абитуриенты`

Основной рабочий лист. Каждый абитуриент выводится один раз. Повторные заявления объединяются по Email.

Основные столбцы:

- `ФИО`;
- `Email`;
- `Телефон`;
- три столбца с баллами;
- `Согласие на зачисление`;
- `Примечание` для ручных записей;
- `Дополнительные квоты`;
- отдельные столбцы бюджетных приоритетов по направлениям.

Приоритеты подсвечиваются так:

- `1` - зеленый;
- `2` - светло-зеленый;
- `3` - желтый.

### `Статистика`

Содержит общие показатели и сводные таблицы по направлениям, квотам и бюджетным приоритетам.

## Запуск проекта через Python

Этот вариант нужен разработчикам или пользователям, которые хотят запускать исходный код.

### Требования

- Windows;
- Python 3.12;
- Git, если проект скачивается командой `git clone`.

### 1. Получите исходный код

Через Git:

```powershell
git clone https://github.com/antonskalkin73/ulstu-reception-convertor.git
cd ulstu-reception-convertor
```

Или скачайте исходный код через кнопку **Code → Download ZIP**, распакуйте архив и откройте PowerShell в папке проекта.

### 2. Создайте виртуальное окружение

```powershell
py -3.12 -m venv .venv
```

### 3. Активируйте окружение

В PowerShell:

```powershell
.\.venv\Scripts\Activate.ps1
```

Если используется обычная командная строка `cmd.exe`:

```bat
.venv\Scripts\activate.bat
```

### 4. Установите зависимости

```powershell
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

Основные зависимости проекта:

- `PyMuPDF` - чтение PDF;
- `openpyxl` - создание и обновление XLSX.

### 5. Запустите программу

Интерактивный запуск:

```powershell
python main.py
```

Далее отвечайте на вопросы так же, как при запуске EXE.

## Запуск с параметрами

Программу можно запускать без интерактивных вопросов.

Создать файл для одного факультета:

```powershell
python main.py --input "C:\Данные\СПИСОК_ДЕКАНАМ.pdf" --faculty "ФИСТ"
```

Создать файлы для всех факультетов:

```powershell
python main.py --input "C:\Данные\СПИСОК_ДЕКАНАМ.pdf" --faculty 0
```

Обновить существующий файл с сохранением примечаний:

```powershell
python main.py `
  --input "C:\Данные\СПИСОК_ДЕКАНАМ.pdf" `
  --faculty "ФИСТ" `
  --existing "C:\Данные\Абитуриенты_ФИСТ_2026-07-09.xlsx"
```

Указать другой каталог для новых файлов:

```powershell
python main.py `
  --input "C:\Данные\СПИСОК_ДЕКАНАМ.pdf" `
  --faculty "ФИСТ" `
  --output-dir "C:\Данные\Готовые списки"
```

Короткие варианты параметров:

```text
-i  путь к PDF
-f  факультет
-e  существующий XLSX
-o  каталог для новых файлов
```

Посмотреть встроенную справку:

```powershell
python main.py --help
```

При запуске с параметрами программа не ждет нажатия клавиши после завершения.

## Возможные ошибки

### `Файл не найден`

Проверьте путь, имя файла и расширение. Для исходных данных требуется PDF, для обновления требуется XLSX.

### `Факультет не найден`

Проверьте код факультета. Программа покажет коды факультетов, найденные в PDF.

### Не удается обновить XLSX

Закройте файл в Excel и повторите запуск. Также проверьте, что у пользователя есть право записи в папку.

### Нет листа `Абитуриенты`

Для обновления нужно выбирать XLSX, ранее созданный этим приложением. Произвольная книга Excel может иметь несовместимую структуру.
