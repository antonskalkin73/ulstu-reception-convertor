# Парсер абитуриентов из PDF на Python
Нужно разработать консольное приложение по извлечению данных из pdf файлов и сохранению в формате xlsx.
В главном меню нужно для начала выбирать вид файла на вход, а затем запрашивать путь к файлу (возможно с кавычками по краям)
Далее могут быть выбраны разные форматы генерации исходных файлов (сохранять в специальный каталог в подкаталог с названием: текущая дата и вид исходного документа)

## Параметры работы
1. Факультет выбирается пользователем при запуске приложения. Если оставить поле пустым или ввести 0, приложение сформирует отдельные файлы для всех факультетов из PDF.
2. Направления подготовки определяются автоматически из исходного PDF.
3. Короткие названия направлений можно настроить в файле `direction_short_names.ini`, который создается рядом с приложением при первом запуске.
4. Квота
    - Бюджет
    - ОтдельнаяКвота
    - ОсобаяКвота
    - Платно
    - Целевая
